# 철학과 김지안 포트폴리오

> "인간의 본질적인 외로움과 그 너머의 연대를 머릿속 상념이 아닌 '**실행**'으로 증명해 나갑니다."

저는 철학이 정적인 학문에 머물지 않고, 타인의 삶에 닿는 구체적인 결과물이 되어야 한다고 믿습니다.

그 첫걸음으로 저는 소설 **≪양진주씨 이야기≫**를 기획하고 출판했습니다.

주인공 진주가 과거의 상처로 고립을 택하면서도 세상의 소리를 수집하며 고독을 견뎌내는 과정은 '인간 존재의 고유성'에 대한 제 철학적 고민이 담긴 결과물입니다.

### AI 연구 질문과 탐구 방법
이 소설을 쓰며 자연스럽게 한 가지 의문을 품게 되었습니다. 

* **탐구 질문:** AI가 인간의 외로움이라는 추상적 감정을 학습할 때, 진주가 찻집에서 마주한 찰나의 '감정적 도약'을 과연 어떠한 문맥적 메커니즘을 바탕으로 이해하고 재현해 낼 수 있는가?

이 질문을 탐구하기 위해 저는 인간 작가가 묘사한 **비언어적 소리**(바람 소리, 찻잔 소리 등)가 주는 정서적 치유 효과를 AI가 어떻게 분류하고 재현하는지 분석해보고 싶습니다. 진정한 연대감은 단순한 정보 전달이 아닌 텍스트 이면의 미세한 분위기에서 오기 때문입니다. 
* **실험 구상:** 예를 들어, AI가 분석한 소리 데이터값이 실제 독자가 느끼는 위로의 감정과 얼마나 일치하는지 비교하는 실험을 구상 중입니다.

### 5년 후 비전
저는 앞으로 이러한 통찰력과 데이터 분석 경험을 바탕으로 기술에 따뜻한 숨결을 불어넣는 '**AI 인문학 콘텐츠 전략가**'가 되고자 합니다. 진주가 세상으로 나아갔듯, 저 또한 AI를 통해 인간의 고독을 어루만지는 새로운 서사를 기획할 것입니다.
